# skill-explorer

## Scope

Analyze a SKILL.md file and generate an interactive HTML visualization with 4 views: foldable overview, structure treemap, code browser with syntax highlighting, and connection graph. Produces a standalone index.html deployable to Cloudflare Pages.

TRIGGERS: Use when:
- User asks to visualize or explore a skill
- User wants to understand a skill's structure
- User asks for a skill map, skill diagram, or skill overview page
- User mentions "skill explorer" or "skill viz"

## Trigger

- Skill trigger: Visualize SKILL.md structure.
- Procedure: see `SKILL.md`.
